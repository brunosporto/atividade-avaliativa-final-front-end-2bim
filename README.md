# StyleWave Moda

Aplicação web de loja de moda desenvolvida com **React 18**, **Tailwind CSS** e **React Router v6**. Permite cadastrar usuários, listar e editar registros, e consultar endereços por CEP. Toda a aplicação roda em um único arquivo HTML, sem necessidade de instalação.

---

## Como Executar

### Opção 1 — Abrir direto no navegador

Baixe o arquivo `index.html` e abra com um duplo clique.

### Opção 2 — Live Server (VS Code)

1. Instale a extensão **Live Server**
2. Clique com o botão direito em `index.html`
3. Selecione **Open with Live Server**

### Opção 3 — Python

```bash
python3 -m http.server 8080
```
Acesse: `http://localhost:8080`

### Opção 4 — Node.js

```bash
npx serve .
```
Acesse: `http://localhost:3000`

---

## Requisitos

- Navegador atualizado (Chrome, Firefox, Edge ou Safari atualizados)
- Conexão com a internet
- Node.js ou Python se quiser usar um servidor local
